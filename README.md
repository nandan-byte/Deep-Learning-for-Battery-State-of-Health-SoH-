# Battery State of Health (SoH) & RUL Prediction

## Overview
This project predicts battery health (SoH) and Remaining Useful Life (RUL) using LSTM models.

## Features
- SoH prediction
- RUL estimation
- Deep Learning (LSTM)
- Frontend UI for visualization

## Tech Stack
- Python
- TensorFlow / Keras
- FastAPI (if used)
- HTML, CSS, JS

## How to Run

### Backend
```bash
cd backend
pip install -r ../requirements.txt
python main.py
